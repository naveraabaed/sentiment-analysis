# 🎬 Movie Review Sentiment Analysis App

## 📌 About Project
This project is an AI-based sentiment analysis system that classifies movie reviews as Positive or Negative using a pre-trained Hugging Face model (DistilBERT).

---

## 🚀 Features
- Positive / Negative sentiment detection
- Confidence score display
- Streamlit web app interface
- IMDB dataset evaluation

---

## 🧠 Model Used
distilbert-base-uncased-finetuned-sst-2-english

---

## ⚙️ How to Run

### 1. Activate virtual environment
venv\Scripts\activate

### 2. Run Streamlit app
streamlit run sentiment_app.py

### 3. Run evaluation script
python real_data.py

---

## 📊 Example Output
- POSITIVE 😊 (99.99%)
- NEGATIVE 😞 (87%)
